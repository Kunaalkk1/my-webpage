Projects

LifeBox - An organ procurement and preservation device

2023 - Present, IISc Bangalore

LifeBox is a medical device that is used to preserve an organ, say heart in this case, for 6 to 8 hours by using PID Algorithm to maintain its temperature and a myriad of non-invasive medical sensors to monitor the heart pH and other parameters to ensure vitality of the heart during transport.

InsuFlo - Insulin Pump for Type-1 Diabetic Patients

2023 - Present, IISc Bangalore

InsuFlo is a highly accurate portable medical insulin pump for Type-1 diabetic patients that dispenses insulin in the patient’s bloodstream on a daily basis. It comes with mobile application assistance for the patient, caregiver and the patient’s doctor.


USB to CAN Converter - Accessory for Battery Management Systems

2023, Inverted Energy

The USB to CAN Converter is a device used to transmit and receive data between the BMS and the PC User Interface of the same. It is also capable of decoding battery data and logging it in a memory card.

Eye Gaze and Squint Prediction and Machine Learning Algorithm

2022, CURIN R&D Project

This device processes images using an ESP32-CAM and sends it to a webserver, where a python algorithm processes the image and generates a dataset. This dataset is used to detect squint using machine learning.

Water Quality Analysis and Prediction - Data Acquisition Robot

2021, B.E. Engineering Project

The obstacle avoiding robot that collects samples from different parts of a waterbody and uses industrial hydroponic sensors to acquire data of the water quality. It sends the data to a server for storage and analysis.

Multi-controller communication (ESP-NOW) to control traffic lights

2021, Smart Indian Hackathon Project

This project was a finalist in Smart India Hackathon 2022, uses a secure and encrypted Wi-Fi protocol on ESP32 to transmit data and gives control of the flow of traffic of any signal to the assigned police officer.

Web-based Hospital Automation System on Google Firebase

2020, B.E. Engineering Project

This is a web portal that connects an entire hospital to a database, monitors every room and patient using sensors connected to ESP8266 and also manages - assignment of doctors, schedules and appointments, emergency alarms and notifications.

IoT-based Smart Farming System on Google Firebase

2020, B.E. Engineering Project

This project can provide a farmer access to the farm weather and soil conditions monitored on ESP32 and provides a web-dashboard hosted on Google Firebase, and also controls the motors for the irrigation system.


Simple Home Automation System using Amazon Alexa

2019, B.E. Engineering Project

This is a basic home automation system that uses relays to power electric appliances, lights, fans, etc. programmed on Arduino IoT Cloud and uses Alexa voice commands to control lights, fan speeds, and more.
