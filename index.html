USB to CAN Converter - Accessory for Battery Management Systems

2023, Inverted Energy Pvt Ltd

The USB to CAN Converter is a device used to transmit and receive data between the BMS and the PC User Interface of the same. It is also capable of decoding battery data and logging it in a memory card.

Eye Gaze and Squint Prediction and Machine Learning Algorithm

2022, CURIN R&D Project

This project of Chitkara University Research & Innovation Netwoek (CURIN) processes video footage in a webserver, where a python algorithm processes the image and generates a dataset. This dataset is used to detect squint using ML.

Multi-controller communication (ESP-NOW) to control traffic lights

2021, Smart Indian Hackathon

This project was a finalist (Top 5) in Smart India Hackathon 2022, uses a secure and encrypted Wi-Fi protocol on ESP32 to transmit data and gives control of the flow of traffic of any signal to the assigned police officer.

Web-based Hospital Automation System on Google Firebase

2020, B.E. Engineering Project

This is a web portal that connects an entire hospital to a database, monitors every room and patient using sensors connected to ESP8266 and also manages - doctors, schedules and appointments, emergency alarms and notifications.

IoT-based Smart Farming System on Google Firebase

2020, B.E. Engineering Project

This project can provide a farmer access to the farm weather and soil conditions monitored on ESP32 and provides a web-dashboard hosted on Google Firebase, and also controls the motors for the irrigation system.
